Optional mergeWhitespaces

mergeWhitespaces?: boolean | "never"

Merge whitespace tokens to saving extra <span>.

When set to true, it will merge whitespace tokens with the next token. When set to false, it keep the output as-is. When set to never, it will force to separate leading and trailing spaces from tokens.

structure?: "inline" | "classic"

The structure of the generated HAST and HTML.
